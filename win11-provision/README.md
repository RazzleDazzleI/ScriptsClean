# Win11 Provision Project

## Structure
- provision.py : main runner
- manifest.yaml : config
- assets/scripts : PowerShell helpers (e.g. RemoveApps.ps1)
- assets/installers : offline installers
- assets/office : ODT + configs
- assets/drivers : drivers (.inf)
- assets/Background : wallpapers, branding
- modules : Python helper modules
- logs : execution logs

Run \provision.py\ to start provisioning.
